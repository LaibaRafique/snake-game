# 🐍 Snake Game in Python

A classic Snake Game built using Python's built-in `turtle` module. Control the snake, collect food to grow longer, increase your score, and avoid colliding with the walls or your own body.

---

## 📌 Features

- 🎮 Classic Snake gameplay
- 🍎 Randomly generated food locations
- 📈 Live score and high score tracking
- ⚡ Game speed increases as the snake grows
- 🟩 Snake body grows after eating food
- 🚧 Game resets when the snake hits the border
- 💥 Collision detection with the snake's own body
- 🎨 Simple graphical interface using Turtle Graphics

---

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics
- Random Module
- Time Module

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/LaibaRafique/snake-game.git
```

### 2. Open the project folder

```bash
cd snake-game
```

### 3. Run the game

```bash
python snake_game.py
```

> Make sure Python 3 is installed on your computer.

---

## 📖 Game Rules

- Eat the red food to grow longer.
- Every food collected adds **10 points**.
- The snake moves faster as your score increases.
- The game ends if:
  - You hit the border.
  - You collide with your own body.
- Your **High Score** is saved during the current game session.
